4.2.15.3 Register Employees

Description

This page allows the user to be able to register employees on the system so they may have authorised access to the system. This action is performed by the administrator/owner/personel with high level authority.

Administrator Registration

 

Reference:

Control:

Functionality Description:

1

First Name Input Box

Click here to enter the first name.

 

2

Last Name Input Box

Click here to enter last name.

3

Email address input box

Click here to enter the email address.

4

Identity Number input box

Click here to enter identity number

5

Cell phone number input box

This function allows the user to add their phone number.

6

Address Input box

The user enters their address here

7

Gender dropdown

Click here to select the gender

8

Username textbox

This function enables the user to have a profile created for them which will allow them access to the system.

9

Password

The user enters their desired password which will allow them be used to verify against the username in the database when they login

10

Add button

This function allows for the employee to be registered on the system.

10

Cancel button

This button terminates the operation and returns the user to the previous page.

•         Possible Error Messages:

o   First name input box → First name is required

o   Last Name input box → Last name is required

o   Cell phone number input box → Cell phone number is required

o   Email Address input box → Email Address is required

o   Identity number input box → Identity number is required
